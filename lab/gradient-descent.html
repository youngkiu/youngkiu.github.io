경사하강법 — Gradient Descent

2차 방정식을 미분하면 1차 방정식이 됩니다. 임의 지점에서 시작해 미분값(기울기)의 반대 방향으로 조금씩 이동하면 전역 최솟값에 도달합니다.

f(x) = ax² + bx + c
→
f′(x) = 2ax + b
→
x ← x − α·f′(x)
