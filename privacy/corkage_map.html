콜키지 맵 개인정보처리방침

최종 업데이트: 2026년 4월 12일

1. 개요

콜키지 맵(이하 "본 앱")은 콜키지 프리 음식점을 쉽게 찾고 등록할 수 있도록 돕는 지도 기반 서비스입니다. 본 개인정보처리방침은 본 앱이 어떤 정보를 수집하고, 어떤 목적으로 이용하며, 어떻게 보호하는지 안내합니다.

2. 수집하는 개인정보 항목 및 이용 목적

본 앱은 다음과 같은 개인정보를 수집할 수 있습니다.

가. 카카오 로그인 정보

  • 수집 항목: 카카오 로그인을 통해 발급되는 사용자 식별자 (카카오 회원번호)
  • 이용 목적: 콜키지 정보 등록·수정·삭제 등 본인 인증이 필요한 기능 제공
  • 수집 시점: 사용자가 등록·수정·삭제 등의 인증 필요 기능을 이용하기 위해 직접 로그인할 때

나. 위치 정보

  • 수집 항목: 정확한 위치(ACCESS_FINE_LOCATION, GPS 기반) 및 대략적인 위치(ACCESS_COARSE_LOCATION)
  • 이용 목적: "내 위치" 버튼 탭 시 사용자 주변의 콜키지 정보를 지도에 표시
  • 수집 시점: 사용자가 직접 "내 위치" 버튼을 탭한 경우에만 (포그라운드 한정, 백그라운드 위치 권한은 요청하지 않음)
  • 저장 여부: 위치 정보는 마지막 지도 중심 좌표를 기기 내부에 캐싱할 뿐이며, 외부 서버로 전송되지 않습니다.

다. 사용자 등록 콜키지 정보

  • 수집 항목: 사용자가 직접 입력한 음식점 이름·주소·콜키지 상태(프리/유료/조건부/반입불가)·콜키지 비용·반입 가능 주류·요리 종류·메모 등
  • 이용 목적: 다른 사용자에게 콜키지 정보 공유
  • 저장 위치: 본 앱 운영자가 직접 운영하는 백엔드 서버

라. 인증 토큰

  • 수집 항목: 서버에서 발급한 인증 토큰
  • 이용 목적: 로그인 세션 유지 및 인증이 필요한 기능 이용
  • 저장 위치: 사용자 기기의 보안 저장소. 외부 서버로 별도 전송되지 않으며, 만료 시 자동 삭제됩니다.

마. 닉네임 (선택)

  • 수집 항목: 사용자가 마이페이지에서 직접 입력한 표시 이름 (1~20자)
  • 이용 목적: 사용자가 등록한 콜키지 정보 옆에 표시되는 작성자명
  • 참고: 닉네임을 별도로 설정하지 않으면 카카오 프로필의 이메일/이름 등에서 자동으로 결정됩니다.

3. 개인정보의 제3자 제공 및 처리위탁

본 앱은 사용자의 개인정보를 본 앱의 정상적인 동작에 필수적인 범위 외에는 제3자에게 제공하지 않습니다. 다만 다음 외부 서비스와의 연동이 있습니다.

  • 카카오(Kakao): 카카오 로그인 인증. 카카오 개인정보처리방침
  • 카카오 지도(Kakao Map): 지도 타일 표시 및 주소·키워드 검색
  • 백엔드 서버: 콜키지 정보의 등록·조회·수정·삭제 기능 제공. 본 앱 운영자가 직접 운영합니다.

4. 개인정보의 보유 및 이용 기간

  • 인증 토큰: 세션 만료 시 또는 사용자가 로그아웃할 때 자동 삭제
  • 위치 캐시: 앱 삭제 시 함께 제거
  • 등록한 콜키지 정보: 사용자가 직접 삭제하거나 탈퇴 요청 시까지 보관 (단, 다른 사용자에게 공유된 정보의 특성상, 사용자가 본인이 등록한 항목을 직접 삭제할 수 있습니다.)

5. 사용자의 권리와 행사 방법

사용자는 본 앱 내에서 다음과 같은 권리를 행사할 수 있습니다.

  • 로그아웃: 우측 상단 프로필 아이콘 → 메뉴에서 로그아웃 → 기기에 저장된 인증 토큰 즉시 삭제
  • 본인 등록 항목 삭제: 콜키지 상세 패널에서 휴지통(🗑️) 아이콘 탭 → 본인이 등록한 항목만 서버에서 삭제 가능
  • 본인 등록 항목 수정: 콜키지 상세 패널에서 연필(✏️) 아이콘 탭 → 본인이 등록한 항목만 수정 가능
  • 닉네임 변경: 우측 상단 프로필 아이콘 → 메뉴에서 닉네임 변경
  • 위치 권한 철회: 기기의 시스템 설정에서 본 앱의 위치 권한 변경 가능
  • 완전한 삭제: 앱 삭제 시 기기 내 모든 캐시·토큰 즉시 제거. 서버에 등록한 항목 일괄 삭제는 아래 연락처로 요청

6. 위치 정보 이용 안내

본 앱은 사용자의 명시적 액션("내 위치" 버튼 탭) 시에만 GPS 기반의 정확한 위치 정보를 수집하며, 백그라운드에서는 일체 위치 정보를 수집하지 않습니다. 정확한 위치는 사용자 주변 콜키지 음식점을 정밀하게 표시하기 위해 사용됩니다.

사용자는 언제든지 기기의 시스템 설정에서 본 앱의 위치 정보 접근 권한을 변경하거나 거부할 수 있습니다.

7. 개인정보의 안전성 확보 조치

  • 인증 토큰은 운영체제가 제공하는 보안 저장소에 저장됩니다.
  • 백엔드 서버와의 통신은 HTTPS로 암호화됩니다.
  • 서버는 사용자의 카카오 회원번호를 그대로 노출하지 않으며, 다른 사용자에게 보이는 작성자명은 마스킹/닉네임 처리 후 표시됩니다.

8. 만 14세 미만 아동의 개인정보

본 앱은 만 14세 미만 아동을 주된 대상으로 하지 않습니다. 본 앱은 카카오 계정 정책에 의존하므로, 카카오 계정 가입 가능 연령 이상의 사용자만 인증 기반 기능을 사용할 수 있습니다.
