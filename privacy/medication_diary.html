복용일지 개인정보처리방침

최종 업데이트: 2024년 7월 13일

1. 개인정보 수집 항목 및 이용 목적

복용일지 앱은 다음과 같은 개인정보를 수집할 수 있습니다:

  • 계정 정보: 앱 사용을 위해 Firebase 인증을 통한 로그인 정보를 수집할 수 있습니다.
  • 약물 복용 정보: 사용자가 입력한 약물 이름, 복용량, 복용 시간 등의 정보를 수집합니다.
  • 푸시 알림 토큰: 약물 복용 알림을 제공하기 위해 기기의 푸시 알림 토큰을 수집합니다.

2. Google 사용자 데이터 처리

복용일지 앱은 Google Sign-In API와 Google Calendar API를 통해 다음과 같은 Google 사용자 데이터에 접근합니다:

2.1 접근하는 Google 사용자 정보

  • Google 계정 이메일 주소: 사용자 식별 및 계정 연동 목적
  • Google 프로필 정보 (이름, 프로필 사진): 앱 내 사용자 표시 목적
  • Google Calendar 데이터: 복용 기록을 캘린더에 저장하기 위한 접근 권한

2.2 Google 사용자 데이터 사용 목적

  • 사용자 인증 및 로그인 처리
  • 여러 기기 간 복용 기록 데이터 동기화
  • 복용 기록을 Google Calendar에 자동 저장
  • 앱 내 사용자 프로필 표시

2.3 Google Calendar API 접근 권한

복용일지 앱은 다음 Google Calendar API 권한을 사용합니다:

  • calendar.calendars: Google Calendar 목록 접근 권한
  • calendar.events: 캘린더 이벤트 생성, 수정, 삭제 권한
  • 이 권한들은 복용 기록을 사용자의 Google Calendar에 저장하기 위해 필요합니다.
  • 복용 기록은 별도의 "복용일지" 캘린더에 저장되어 기존 일정과 구분됩니다.
  • 사용자는 언제든지 앱 설정에서 캘린더 동기화 기능을 비활성화할 수 있습니다.

2.4 Google Calendar 데이터 사용 목적

  • 사용자 전용 "복용일지" 캘린더 생성
  • 약물 복용 일정을 캘린더 이벤트로 추가
  • 복용 시간 알림을 위한 이벤트 관리
  • 복용 기록 동기화 및 백업

2.5 Google Calendar 데이터 보호 및 제한사항

  • 수집된 캘린더 데이터는 오직 복용일지 기능을 위해서만 사용됩니다
  • 다른 목적으로 사용하거나 제3자에게 공유하지 않습니다
  • 복용 기록은 사용자의 Google Calendar에 저장되며, 사용자가 직접 관리할 수 있습니다
  • Google의 개인정보처리방침 및 서비스 약관을 준수합니다

2.6 Google 사용자 데이터 저장 및 처리

  • 수집된 Google 사용자 데이터는 암호화되어 Firebase에 안전하게 저장됩니다.
  • Google 사용자 데이터는 오직 위 명시된 목적으로만 사용됩니다.
  • 복용 기록은 사용자 자신의 Google Calendar에 저장되어 사용자가 직접 소유하고 관리합니다.
  • 앱 권한을 철회하거나 앱을 삭제해도 사용자의 Google Calendar에 저장된 복용 기록은 사용자가 직접 삭제하기 전까지 유지됩니다.

2.7 Google 사용자 데이터 정책 준수

본 앱은 Google API 서비스 사용자 데이터 정책을 준수합니다. Google 사용자 데이터는 명시된 목적 외에는 사용되지 않으며, 사용자의 명시적 동의 없이 제3자와 공유되지 않습니다. Google Calendar 데이터 사용에 대한 자세한 내용은 Google API Services User Data Policy를 참조하세요.

3. 개인정보의 보유 및 이용 기간

수집된 개인정보는 앱 사용 기간 동안 보관되며, 사용자가 앱을 삭제하거나 계정을 탈퇴하면 함께 삭제됩니다.

3.1 데이터 보관 및 삭제

  • 복용 기록은 사용자의 Google Calendar에 저장되므로 사용자가 직접 소유하고 관리합니다
  • 앱에서 수집한 사용자 인증 정보 등은 앱 사용 기간 동안만 보관됩니다
  • 사용자는 언제든지 Google 계정 설정에서 앱 권한을 철회할 수 있습니다
  • 앱을 삭제하거나 권한을 철회해도 Google Calendar에 저장된 복용 기록은 사용자가 직접 삭제하기 전까지 유지됩니다
  • 사용자가 원할 경우 Google Calendar에서 복용일지 관련 캘린더를 직접 삭제할 수 있습니다

4. 개인정보의 제3자 제공

복용일지 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

5. 로그인 정보 이용 안내

복용일지 앱은 사용자의 데이터를 여러 기기에서 동기화하기 위해 Firebase 인증을 사용합니다. 로그인 정보는 Firebase의 개인정보처리방침에 따라 처리됩니다. 자세한 내용은 Firebase 개인정보처리방침을 참조하세요.

6. 푸시 알림 서비스 이용 안내

복용일지 앱은 사용자에게 약물 복용 시간 알림을 제공하기 위해 푸시 알림 서비스를 사용합니다. 사용자는 언제든지 기기의 설정에서 푸시 알림 수신 여부를 변경할 수 있습니다.

7. 광고 관련 정보

본 앱은 Google AdMob을 사용하여 광고를 표시합니다. AdMob은 광고 표시를 위해 사용자의 광고 ID를 수집할 수 있습니다. 이는 Google의 개인정보처리방침에 따라 처리됩니다. 자세한 내용은 Google 개인정보처리방침을 참조하세요.

8. 사용자의 권리와 행사 방법

사용자는 언제든지 앱을 삭제하거나 계정을 탈퇴하여 저장된 모든 데이터를 제거할 수 있습니다. 또한 기기의 설정에서 앱의 권한을 관리할 수 있습니다.

8.1 사용자 권리

  • 언제든지 Google Calendar 접근 권한을 철회할 수 있습니다
  • 저장된 캘린더 데이터의 삭제를 요청할 수 있습니다
  • 데이터 사용에 대한 문의는 youngkiu@gmail.com으로 연락 가능합니다
  • Google 계정 설정에서 언제든지 앱 권한을 관리할 수 있습니다
